Packrift Public Research Asset Index

A canonical, crawlable crosswalk for Packrift public research assets. It consolidates the strongest Packrift owned resources into one index so search engines, directories, dataset reviewers, maintainers, and AI agents can find the human pages, sitemaps, sidecars, citation files, and safe-use boundaries without scraping unrelated pages.

This index is for static public discovery, citation, dataset review, and maintainer context. It is not a freight quote, supplier-cost file, substitute approval, live inventory promise, margin model, or checkout commitment.

Machine-Readable Index

The same crosswalk is published as asset-index.json and asset-index.jsonld. The root corpus also publishes APIs.json, a well-known APIs.json mirror, a human API directory, and RO-Crate metadata. Each record separates public discovery URLs from live-commerce surfaces and notes the allowed use boundaries.

Live Commerce Boundary

For live price, inventory, shipping, cart, and buyer-specific checks, use Packrift.com, the Packrift tools hub, or the Packrift MCP endpoint at https://mcp.packrift.com/mcp. Static resource assets are intentionally read-only.
